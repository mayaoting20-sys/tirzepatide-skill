# 替尔泊肽注射液说明书

## 基本信息

| 项目 | 内容 |
|------|------|
| 通用名 | 替尔泊肽注射液 |
| 商品名 | Mounjaro（糖尿病适应症）/ Zepbound（减重适应症）|
| 英文名 | Tirzepatide Injection |
| 作用机制 | 双靶点：GIP受体激动剂 + GLP-1受体激动剂 |
| 剂型 | 预充式注射笔，皮下注射 |
| 规格 | 2.5mg、5mg、7.5mg、10mg、12.5mg、15mg（每支0.5mL）|
| 注射频率 | 每周一次 |
| 上市公司 | 礼来公司（Eli Lilly）|

---

## 适应症

1. **2型糖尿病**：配合饮食和运动，用于改善成人2型糖尿病患者的血糖控制
2. **慢性体重管理**：用于初始BMI≥30，或BMI≥27且合并至少一种体重相关合并症（如高血压、2型糖尿病、血脂异常等）的成年人

---

## 用法用量

### 起始剂量
- 起始剂量：**2.5mg，每周一次**，皮下注射
- 起始4周内维持2.5mg，让身体适应，减少胃肠道不良反应

### 剂量递增方案

| 周数 | 剂量 |
|------|------|
| 第1-4周 | 2.5mg/周 |
| 第5-8周 | 5mg/周 |
| 第9-12周 | 7.5mg/周（如需继续增量）|
| 第13-16周 | 10mg/周（如需继续增量）|
| 第17-20周 | 12.5mg/周（如需继续增量）|
| 第21周起 | 15mg/周（最大剂量）|

> 注意：如患者对当前剂量耐受良好，可按上述方案递增；如出现不耐受，可维持当前剂量或降回上一剂量。

### 注射部位
- 腹部（脐周5cm以外）
- 大腿前外侧
- 上臂外侧
- 每次注射需轮换部位，避免在同一位置重复注射

### 注射方法
1. 从冰箱取出后，室温放置30分钟再注射
2. 检查药液是否清澈（允许轻微黄色），无颗粒
3. 选择注射部位，用酒精棉片消毒
4. 捏起皮肤，90度角刺入，按下按钮
5. 听到"咔哒"声后保持5秒，再移开注射笔
6. 废弃注射笔放入专用利器盒

---

## 储存条件

- 未开封：冷藏保存（2-8°C），避免冷冻
- 开启后：可在室温（≤30°C）保存最长21天
- 避免阳光直射
- 不可冷冻，冷冻后不可使用

---

## 常见不良反应

### 胃肠道反应（最常见，通常随时间减轻）
| 反应 | 发生率 | 说明 |
|------|--------|------|
| 恶心 | 约30-35% | 多在起始或增量时出现，2-4周后减轻 |
| 腹泻 | 约20-25% | |
| 呕吐 | 约15-20% | |
| 便秘 | 约10-15% | |
| 腹痛/腹胀 | 约10% | |
| 消化不良 | 约8% | |

### 其他反应
- 注射部位反应（红肿、瘙痒）：约3-5%
- 疲劳感：约5%
- 头晕：约5%
- 脱发（体重快速下降相关）：部分用户报告

### 严重不良反应（罕见）
- 急性胰腺炎（出现持续腹痛需立即就医）
- 胆囊疾病（胆结石风险略升高）
- 心率加快（平均增加约2-4次/分钟）
- 严重过敏反应

---

## 禁忌症

- 有甲状腺髓样癌（MTC）个人史或家族史
- 多发性内分泌腺瘤病2型（MEN 2）患者
- 对替尔泊肽或任何辅料过敏
- 妊娠期（建议停药至少2个月后再备孕）

---

## 注意事项

1. **低血糖风险**：单独使用时低血糖风险低，但与胰岛素或磺脲类药物合用时风险增加
2. **胰腺炎**：有胰腺炎病史者慎用，出现持续严重腹痛立即停药就医
3. **糖尿病视网膜病变**：快速改善血糖可能导致视网膜病变暂时加重
4. **肾功能**：严重肾功能损害患者需谨慎，可能加重脱水相关肾损伤
5. **口服药物**：替尔泊肽减缓胃排空，可能影响口服药物吸收速度

---

## 漏打处理

- 若距下次注射时间**≥4天**：尽快补打
- 若距下次注射时间**<4天**：跳过，按原计划下次注射
- 不可在同一天注射两次

---

## 与其他药物相互作用

- **口服避孕药**：替尔泊肽可能降低口服避孕药效果，建议同期使用额外避孕措施
- **华法林**：开始或调整替尔泊肽剂量时，需更频繁监测INR
- **胰岛素/磺脲类**：合用时注意低血糖风险，可能需减少其剂量
