# 替尔泊肽不同体重基数人群的降重效果

## 核心临床数据来源

主要来自 **SURMOUNT 系列临床试验**（礼来公司赞助，全球多中心随机对照试验）：
- SURMOUNT-1：2539名非糖尿病肥胖/超重成人，用药72周
- SURMOUNT-2：938名合并2型糖尿病的肥胖/超重成人，用药72周
- SURMOUNT-4：停药后反弹观察试验

---

## 按剂量的平均减重效果（SURMOUNT-1，72周）

| 剂量 | 平均体重下降比例 | 平均绝对体重下降 |
|------|----------------|----------------|
| 安慰剂 | -3.1% | 约-3.5kg |
| 5mg | -15.0% | 约-15.7kg |
| 10mg | -19.5% | 约-19.9kg |
| 15mg | -20.9% | 约-22.5kg |

> 基线平均体重约107kg，平均BMI约38

## 按体重基数分层预测效果

### 轻度超重（BMI 27-30，体重约70-85kg）

**典型人群：** 身高165cm，体重75kg，BMI 27.5

| 剂量 | 预期减重比例 | 预期减重量 | 72周后体重 |
|------|------------|-----------|-----------|
| 5mg | 12-16% | 9-12kg | 63-66kg |
| 10mg | 16-20% | 12-15kg | 60-63kg |
| 15mg | 18-22% | 13.5-16.5kg | 58-61kg |

**注意：** 起始体重较低者，绝对减重量通常少于高体重者，但比例相近。部分研究显示低BMI起点者药物效果略低。

## 合并2型糖尿病人群（SURMOUNT-2数据）

糖尿病患者减重效果略低于非糖尿病患者，原因包括：部分降糖药本身有增重效应、胰岛素抵抗影响等。

| 剂量 | 平均减重比例（糖尿病人群） |
|------|--------------------------|
| 5mg | -12.8% |
| 10mg | -14.7% |
| 15mg | -15.7% |

## 真实世界数据补充（非临床试验）

临床实践中观察到的效果分布（来自多个真实世界研究和用户报告）：

| 减重效果 | 人群比例 |
|---------|---------|
| 减重>20%体重 | 约35-40%（15mg组）|
| 减重15-20%体重 | 约25-30% |
| 减重10-15%体重 | 约20-25% |
| 减重5-10%体重 | 约10-15% |
| 减重<5%（低/无反应） | 约10-15% |

## 减重时间线（典型进程）

| 时间节点 | 平均累计减重（15mg组）|
|---------|---------------------|
| 4周 | 约-2% |
| 8周 | 约-4% |
| 12周（完成基础加量）| 约-7% |
| 24周（6个月）| 约-13% |
| 36周（9个月）| 约-18% |
| 52周（1年）| 约-20% |
| 72周（约1.5年）| 约-22.5% |

> 减重速度：前6个月较快，之后逐渐放缓进入平台期，约52-72周达到稳定。

## 影响个人减重效果的关键因素

**效果更好的因素：**
- 非糖尿病人群
- 起始BMI较高
- 严格控制饮食热量
- 配合规律运动（尤其抗阻训练）
- 达到更高剂量（10mg/15mg）
- 坚持用药不中断
- 睡眠质量好

**效果较差的因素：**
- 合并2型糖尿病
- 同时使用胰岛素或其他增重药物
- 饮食未作调整
- 代谢率偏低（甲状腺功能低下等）
- 压力大、睡眠差（皮质醇升高拮抗减重）
- 基因多态性（GLP-1受体基因变异）

## 停药后体重变化（SURMOUNT-4数据）

用药52周后停药，观察52周：

| 时间 | 平均体重变化 |
|------|------------|
| 停药时（基线）| 已减重约20% |
| 停药后20周 | 反弹约约7%（约回到-13%）|
| 停药后52周 | 反弹约约12%（约回到-8%）|

> 停药后平均回弹约2/3的减重效果，1/3的减重效果得以保持（若生活方式改变持续的话）。
